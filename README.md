**Student**: Pavel Prokhorov
**Semester**: Fall 2025
**Instructor**: Prof. Anna Baynes

## Project Overview
This project focused on developing cybersecurity skills through a combination of:

- Completing the official ISC2 "Certified in Cybersecurity" (CC) online training program
- Participating in the BuckeyeCTF 2025 cybersecurity hackathon
- Styding for and taking the ISC2 CC certification exam

The goal of the project was to gain a solid understading of Cybersecurity principals and hands-on
expereince in cybersecurity fundamentals.
